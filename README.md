# 🔗 Website Link Checker

A Python tool that scans a website, extracts all links, checks whether each link is working or broken, and exports the results.

## Features

- Download a webpage
- Extract all links
- Convert relative URLs to absolute URLs
- Check each link
- Export results to:
  - CSV
  - JSON
  - Excel (.xlsx)

## Technologies

- Python
- Requests
- BeautifulSoup4
- openpyxl
